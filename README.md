# pdd-parent
仿拼多多的购物小程序（后端）

### 技术选型：

* SpringBoot

* SpringCloud：Spring Cloud Gateway、Spring Cloud 、OpenFeign、Spring Cloud Alibaba Nacos

* MyBatis-Plus

* Redis

* Redisson

* RabbitMQ

* ElasticSearch +Kibana

* ThreadPoolExecutor

* OSS

* Knife4j（Swagger）

* Nginx

* MySQL

* 微信支付

* 微信小程序

* Docker

* DockerFile
